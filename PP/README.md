# 3dgp-ex-plague-apocalypse

Wat testen: Kaarten mechanic



Regels:
Elke beurt krijg jeeen keuze tussen 3 random kaarten. Deze kaart gebruik op het begin van uw beurt.

Harte (1-7): pion extra leven, maar mag niet meer bewegen tot extra leven weg is
Harte (8+): pion reviven, volgende beurt maar 2 kaarten kiezen ipv 3
Ruit(1-7): 2 keer bewegen achter elkaar met 1 pion, pion mag volgende beurt niet gebruikt worden
Ruit (8+): pion word koninging voor 1 beurd, pion gaat volgende beurt dood.
